# SEO: Step 0 — Query → Region → Landing Page Map

Цель шага 0: чтобы у каждого "денежного" запроса была понятная "целевая страница",
которую мы будем продвигать и оптимизировать под этот запрос (а не пытаться
ранжироваться одной главной под всё).

Формула:
`запрос (что ищут) → регион (где ищут) → целевая страница (куда ведем трафик)`

## Регионы (как вы описали)

- Основные области: Воронежская, Тамбовская, Курская, Белгородская, Липецкая

## Что такое "целевая страница"

Пример: запрос "зерносушилка под ключ".

- Плохо: пытаться ранжировать главную `/` (слишком общий интент, мало текста по теме, трудно расширять).
- Лучше: отдельная страница услуги `/services/grain-dryers` с:
  - заголовком (H1) под запрос
  - описанием работ, этапов, комплектаций
  - кейсами (фото/объекты)
  - FAQ
  - CTA: заявка/консультация

## Что я могу сделать сам после утверждения этой карты

1) Создать эти страницы в роутинге (React Router) и добавить SEO-мета под каждую.
2) Обновить `public/sitemap.xml`, чтобы новые URL попадали в индекс.
3) Добавить на страницы структуру блоков (H1/H2, FAQ, кейсы, лид-форма).
